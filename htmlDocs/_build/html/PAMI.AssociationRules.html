Bases: object

Description

RuleMiner is an algorithm to mine association rules

Reference

Parameters

  • frequentPattern – file or dataframeb : File containing all the frequent patterns.

  • measure – str : condition to calculate the strength of rule

  • threshold – float or int or str : condition to satisfy

Methods to execute code on terminal

Format:
>>>  python3 RuleMiner.py <inputFile> <outputFile> <measure> <threshold>
Example:
>>>  python3 RuleMiner.py sampleTDB.txt rules.txt 'lift' 0.5

Note

measure can be lift or leverage or confidence
